# Installation
```sh-session
$ npm i @prakash106/got-quotes
```

# Usage

#### To get the random quote of game of throne.
```sh-session
$ got quote
```
 

#### Character wise quotes
```sh-session
$ got quote [character_name]
```

#### List of all Characters
```sh-session
$ got characters
```
